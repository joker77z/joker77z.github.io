회사에 공유하기 위해 신뢰할 수 있는 출처에 있는 내용을 가공해서 정리한 내용입니다.

구글의 코어 웹 바이탈은 웹 페이지 로딩 속도, 모바일 친화성, 세이프 브라우징, 암호화(HTTPS적용 여부), 방해요소 여부 등과 같은 웹 콘텐츠 이용자의 사용자 경험을 측정하는 많은 요인 중에서 구글이 중요하다 강조하는 3가지 메트릭스 LCP(Largest Contentful Pain), FID(First Input Delay), CLS(Cumulative Layout Shift)를 중요시 하는 개념입니다.

  • LCP는 페이지의 주요 컨텐츠가 로드 되었을 때 타임입니다.

왜 페이지 성능이 중요할까?

페이지 로드 시간이 길수록 이탈률에 심각한 영향을 미칩니다.

  • 페이지 로드 시간이 1초에서 3초로 늘어나면 이탈률은 32% 증가합니다.
  • 페이지 로드 시간이 1초에서 6초로 늘어나면 이탈률은 106% 증가합니다.
