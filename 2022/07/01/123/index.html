왜 페이지 성능이 중요할까?

페이지 로드 시간이 길수록 이탈률에 심각한 영향을 미칩니다.

  • 페이지 로드 시간이 1초에서 3초로 늘어나면 이탈률은 32% 증가합니다.
  • 페이지 로드 시간이 1초에서 6초로 늘어나면 이탈률은 106% 증가합니다.
